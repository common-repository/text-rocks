== Description ==

Our plugin is an omni-channel chat widget. It would enable the site owners to link their fb messenger, whatsapp, livechat accounts to our widget from our platform. Site visitors can choose from any of these options to contact the site owner.

== Installation ==

The quickest method for installing Text.Rocks is:

1. Visit the plugins page within your dashboard and select “Add New”
2. Search for "Text.Rocks";
3. Click "Install Now"
4. Finally click "Activate Plugin"
5. Enter widget ID from text.rocks account.
6. text.rocks account can be created for free on https://text.rocks

If you would prefer to do things manually then follow these instructions:

1. Upload the `textrocks` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Settings -> Textrocks screen, click on WordPress
